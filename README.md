# 📱[EaseEmbro Mobile](https://play.google.com/store/apps/details?id=com.sampurnalabs.easeembro)

**EaseEmbro Mobile** is a factory floor production data entry app to automate the production logging for employees on factory floors.

---

## 🎯 Overview

EaseEmbro is built to solve the complex challenges of managing an embroidery manufacturing business that operates 24/7 with multiple machines, shifts, and employees. The system automates:

- **Production Tracking**: Daily shift-based data entry for each machine
- **Salary Calculations**: Automatic computation of normal duty, extra duty, and bonus payments
- **Employee Management**: Comprehensive employee profiles with role-based configurations
- **Machine Management**: Track machine productivity and maintenance
- **Financial Reports**: Generate detailed salary slips, production reports, and analytics
- **Data Backup & Recovery**: Automated and manual backup capabilities

## ✨ Features

### Core Functionality

| Feature | Description |
|---------|-------------|
| **📊 Dashboard** | Real-time overview of production stats, machine status, and key metrics |
| **📝 Data Entry** | Excel-like spreadsheet interface for fast daily production recording |
| **👥 Employee Management** | Complete employee profiles with multiple role types and salary configurations |
| **🏭 Machine Management** | Configure machines with custom pay rates and track productivity |
| **💰 Salary Management** | Automated monthly salary calculations with detailed breakdowns |
| **📈 Reports & Analytics** | Comprehensive reporting system with PDF/Excel export |
| **⚙️ Configuration** | Fully customizable bonus levels, work areas, and extra duty rates |

### Advanced Features

- **🔄 Multi-Employee Support**: Handle shared shifts, secondary operators, and helper assignments
- **💸 Advance/Loan Management**: Track employee advances with automated monthly deductions
- **📦 Automated Backups**: Scheduled and manual database backups with restore capability
- **🔐 Hardware-Bound Licensing**: Secure license validation tied to hardware fingerprint
- **🔄 Auto Updates**: Automatic application updates with data preservation

## 📦 Installation

1. Download the latest installer from [Releases](https://github.com/om-jasani/EaseEmbro-Releases/releases)
2. Run `EaseEmbro-Setup-x.x.x.exe`
3. Follow the installation wizard
4. Launch EaseEmbro from the Start Menu or Desktop shortcut
5. Activate with your license key

---

## 🔐 License System

EaseEmbro uses a hardware-bound licensing system. When you first run the application, it will generate a unique hardware fingerprint. Please contact support with this ID to receive your license key.
